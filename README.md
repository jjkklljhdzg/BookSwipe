# BookSwipe

Интерактивное веб-приложение для поиска книг через механику свайпов с персонализированными рекомендациями. Проект превращает рутинный поиск литературы в увлекательную игру, где пользователь "свайпает" книги (аналогично Tinder), а система учится на его предпочтениях и предлагает всё более точные рекомендации.

## О проекте

BookSwipe решает проблему "парада выбора" — когда пользователь хочет найти новую книгу, но теряется в огромных каталогах. Вместо бесконечного скроллинга мы предлагаем игровой формат:

- Свайп-лента — оценивайте книги быстро и интуитивно
- Умные рекомендации — система анализирует ваши лайки и подбирает книги по вкусу
- Коллекции — организуйте чтение ("Хочу прочитать", "Читаю", "Прочитано")
- Социальный слой — отзывы, рейтинги, возможность делиться находками

## Установка и запуск

Клонируйте репозиторий:
git clone https://github.com/jjkklljhdzg/BookSwipe

Установите зависимости:
npm install
npm install better-sqlite3

Запустите сервер разработки:
npm run dev

Откройте http://localhost:3000 в браузере.

## Реализованные функции

- Регистрация и авторизация
- Работа с базой данных — чтение книг из SQLite, хранение пользовательских данных
- Рекомендательная система — базовый алгоритм подбора книг на основе жанров и лайков
- Управление коллекциями
- Личный кабинет — редактирование профиля, аватара, персональных данных
- Поиск — быстрый поиск книг по названию и автору

## Аналитическая часть

**Целевая аудитория и проблема**

Проект создан для молодой аудитории (18-30 лет), которая:
- Хочет находить новые книги, но не любит долго искать в больших каталогах
- Ценит персонализацию и игровые элементы

**Архитектурные решения**

1. Свайп-интерфейс — выбран как самый интуитивный и вовлекающий формат для быстрой оценки контента
2. SQLite — идеален для учебного проекта: не требует отдельного сервера, работает из коробки, подходит для MVP
3. Next.js — позволяет быстро создать full-stack приложение с SSR и API routes в одном проекте
